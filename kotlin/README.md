# Filament Kotlin Multiplatform

This workspace contains the Kotlin Multiplatform library that wraps the C ABI from `c-wrapper`.

## Targets in this scaffold

- Android (`androidTarget`)
- iOS (`iosArm64`, `iosSimulatorArm64`)
- macOS (`macosArm64`)
- Linux (`linuxX64`, `linuxArm64`)
- Windows (`mingwX64`)
- Web (`js(IR)`)

## Quick check

```bash
cd kotlin
./gradlew :filament-kmp-api:tasks
```

The Gradle wrapper is checked into this folder.
